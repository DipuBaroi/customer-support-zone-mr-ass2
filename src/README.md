1. What is JSX, and why is it used?

Ans: JSX (JavaScript XML) is a syntax of extention for react that allows us to write HTML -like code inside javaScript. It makes UI code more readable and easier to write.

2. Difference between state and Props?

Ans: Props are passed from parents to child and they are read only. On the other hand State is managed inside the component and can be changed.

3. What is useState hook?

Ans: useState hook is a react hook that allows functional components to manage state. It returns a current state and a function to update it.

4. How to share state between components?

Ans: state is lifting up to the nearest common parent and passed via props. This is called lifting state up.

5. How is event handling done in react?

Ans: Event hanlding is done using camelCase syntax and functions.
